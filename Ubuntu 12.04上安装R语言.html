Ubuntu 12.04上安装R语言

作者：凯鲁嘎吉 - 博客园 http://www.cnblogs.com/kailugaji/

R的安装

sudo gedit  /etc/apt/sources.list

在sources.list文件后面添加一行

deb http://ftp.ctex.org/mirrors/CRAN/bin/linux/ubuntu maverick/

保存并关闭文件

更新软件源

sudo apt-get update

安装软件

sudo apt-get install r-base r-base-dev

安装完成后，输入

wrr@ubuntu:~$ R

出现以下结果则成功

退出R
