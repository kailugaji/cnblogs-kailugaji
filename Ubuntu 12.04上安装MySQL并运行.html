Ubuntu 12.04上安装MySQL并运行

作者：凯鲁嘎吉 - 博客园 http://www.cnblogs.com/kailugaji/

安装MySQL数据库

sudo apt-get update  
sudo apt-get install mysql-server

确认是否安装成功

sudo netstat -tap | grep mysql

当mysql节点处于LISTEN状态表示启动成功

登录数据库

mysql -u root -p

MySQL数据库常用操作

1.显示所有数据库（注意后面一定要有分号，表示一句话的结束）

2.创建新数据库

mysql> create database WRR;

3.创建新的表

mysql> use WRR;
mysql> create table Student(pid varchar(20) primary key, password varchar(20), status int);

4.插入数据

mysql> insert into Student values("2018001","2018001",0);

5.修改数据

（1）修改表名

mysql> alter table Student rename Student1;

（2）修改表中某列的名字

mysql> alter table Student1 change column status status1 int;

（3）修改某一学生的数据

mysql> update Student1 set status1=1 where pid="2018001";

6.删除数据

（1）删除学生数据

mysql> delete from Student1 where pid="2018002";

（2）删除学生表

mysql> drop table Student1;

（3）删除数据库

mysql> drop database WRR;

7.查看MySQL版本

8.退出MySQL

mysql> quit
